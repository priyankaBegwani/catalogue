# Business Context

## Product Purpose

This platform helps ethnic wear businesses:

- manage product catalogs
- organize product variants
- manage retailers
- generate shareable catalogs
- streamline wholesale operations

---

# User Types

## Admin

Capabilities:
- manage retailers
- manage products
- manage categories
- upload catalog assets
- control access

## Retailer

Capabilities:
- browse products
- view catalogs
- access assigned inventory
- use mobile-friendly catalog experience

---

# Product Categories

Examples:
- Kurta
- Sherwani
- Ethnic Sets
- Jackets
- Accessories

---

# UX Priorities

Highest priorities:

1. Fast mobile experience
2. Easy product discovery
3. Clean product presentation
4. Reliable image loading
5. Simple retailer onboarding

---

# Business Constraints

- Many users operate primarily on phones
- Internet quality may vary
- Catalog sharing is a core workflow
- Large image-heavy catalogs are common
- Admin workflows must remain simple

---
